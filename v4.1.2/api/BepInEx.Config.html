EnableConfigWrapperCaching

Improve performance by minimizing reads and writes to the config by ConfigWrapper<T>. This will cause each wrapper to not see changes done to its setting if they were done from outside of that wrapper. Use only if you use a single ConfigWrapper for your settings, or don't use wrappers at all.
